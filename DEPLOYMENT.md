# Deployment Guide

## Vercel

1. Sign up or log in to your Vercel account.
2. Click on the "New Project" button.
3. Import your GitHub repository (funnyhowimdev/freelancer-marketplace).
4. Vercel will automatically detect the framework used.
5. Set the necessary environment variables (if any) in the settings.
6. Click "Deploy" and wait for the build process to complete.
7. Visit the provided URL to see your deployed site.

## Railway

1. Create an account on Railway.
2. Start a new project and select "Deploy from GitHub".
3. Connect your GitHub and select the freelancer-marketplace repository.
4. Configure any required environment variables.
5. Click "Deploy" and wait for the deployment to finish.
6. Access your application using the provided Railway domain.

## Render

1. Sign up for a Render account.
2. Click on the "New" button and select "Web Service".
3. Connect your GitHub and choose the freelancer-marketplace repository.
4. Specify main as the branch to deploy from.
5. Set up any necessary environment variables.
6. Click "Create Web Service" to initiate the deployment.
7. Once deployed, visit your Render domain to access the application.
